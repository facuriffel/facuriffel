### Sobre mí

Soy desarrollador de software en formación, actualmente en mi tercer y ultimo año de una **Tecnicatura en Desarrollo de Software** con foco en construir aplicaciones web y móviles completas - desde la interfaz hasta la base de datos.

Me manejo tanto en **frontend** con React como en **backend** con Java y Spring Boot. Aprendo rápido, trabajo con Git desde el día uno y me importa escribir código que otros puedan entender y mantener.

Estoy buscando mi primera experiencia profesional como **pasante o desarrollador junior**, presencial en Santa Fe o remoto.
